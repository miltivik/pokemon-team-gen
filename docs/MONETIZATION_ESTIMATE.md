## Estimación de Ingresos

### Parámetros

- **RPM (Revenue Per Mille)**: $4.00 por cada 1,000 impresiones
- **Anuncios por sesión**: 5-6
- **CPM efectivo**: $4.00

### Cálculo de Ingresos

```
Ingresos por 1,000 usuarios únicos:

Usuario promedio = 5.5 anuncios/sesión
Impresiones = 1,000 usuarios × 5.5 anuncios = 5,500 impresiones

Ingresos = (5,500 / 1,000) × $4.00 = $22.00 por 1,000 usuarios
```

## Notas

1. **RPM variable**: El RPM real puede variar según:
   - Geolocalización del tráfico
   - Categoría del sitio (gaming tiene buenos CPMs)
   - Estacionalidad
   - Formato de anuncios

2. **Optimización**:
   - Los anuncios de display (banner) típicamente tienen mayor CPM
   - Los anuncios responsivos se adaptan mejor a dispositivos móviles
   - La posición "above the fold" tiene mayor CTR

3. **Recomendaciones**:
   - Monitorear CTR (Click Through Rate) objetivo > 1%
   - Ajustar posiciones si el RPM baja
   - Considerar ads.txt para mayor transparencia
